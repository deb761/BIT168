Style Guide

Fonts

Dancing Script
This font is used for the headers. The font was selected because I wanted the text to have a hint of a handwritten appearance, yet still be slightly formal and easy to read. The font is licensed under the SIL Open Font Licence, 1.1
Lora
This font is used for body text. The font was selected because serif styles tend to be easier to read in large blocks of text, and because the font mimicked the classic curves of Font 1. The license is the SIL Open Font Licence, 1.1

Overall Formatting

The page is set up to show a winter sky with clouds and some blue sky showing through, and a row of snow-covered trees on the bottom. Slightly transparent fields will hold the header, navigation, content, and footer sections.

The navigation bar will have buttons for the main navigation links, with the current page indicated by the button being reverse-highlighted. When hovered over, the navigation buttons are also reverse-highlighted, and when active it will pick up the footer background color.

The overall site colors are cool colors such as blue and green to coordinate with the winter theme of the background. Accents will use the same general hues as the background and text.

The general font-family for the site is Calibri with backup fonts Arial, Helvetica and Sans Serif. The general text size is 1.2em. Headings are in Cambria Bold Italic backup fonts Times New Roman and Serif, and larger font sizes than body text.
